\chapter{Verdict Tables for the Granny Detection}\label{app:granny_verdict_tables}

This appendix tabulates the homological filtering verdicts for all $83$ candidate braid families generated by the train-track analysis of Chapters \ref{chap:stratoriented} and \ref{chap:stratatraintrackanalysis}, evaluated against the granny target Alexander polynomial $\Delta(t) = (t^2-t+1)^2$. The data here support the proof of Theorem \ref{thm:main_detection}, specifically the conclusions of Theorems \ref{thm:candelabra_stratum_classification} and \ref{thm:two_triangle_stratum_classification}.

For each family, we list:
\begin{itemize}
    \item the family name (as recorded in Appendices \ref{app:candidate_braids_candelabra}--\ref{app:candidate_braids_track_17}),
    \item the determinant $\det(I - A_{\text{red}})$, expressed as a constant or a polynomial in the family's twist parameters $k, m, j$,
    \item the characteristic polynomial $P(t) = \det(tI - A_{\text{red}})$,
    \item the verdict, which records the means of elimination.
\end{itemize}

The verdicts are categorized as follows:
\begin{description}
    \item[\textbf{F1}] Filter 1 fails: $\det(I - A_{\text{red}})$ is a constant other than $\pm 1$, so the open book does not bound an integer homology sphere.
    \item[\textbf{F1*}] Filter 1 fails for parametric families: there exist no non-negative integer parameter values for which $|\det(I-A_{\text{red}})| = 1$.
    \item[\textbf{F2}] Filter 2 fails: even at parameter values where Filter 1 admits a solution, the characteristic polynomial does not match the granny target $(t^2-t+1)^2$ at any such value.
\end{description}

All verdicts were computed by direct evaluation of the matrices $M_1, \dots, M_5$ from Section \ref{sec:homological_filtering}, using the parsing rules and execution pipeline described in the implementation of the homological filter. The reduced action is invariant under conjugation, full twists, and mirroring (Section \ref{sec:homological_filtering}), so testing a single representative braid word per family suffices.

\section{Summary}

Across all $83$ candidate families:
\begin{itemize}
    \item $51$ families are eliminated by Filter 1 alone, with $\det(I-A_{\text{red}})$ a constant other than $\pm 1$.
    \item $20$ families are eliminated by Filter 1*, where $\det(I-A_{\text{red}})$ is parametric and admits no non-negative integer solutions to $|\det| = 1$.
    \item $12$ families pass Filter 1 at some non-negative integer parameter values, but at every such value the characteristic polynomial $P(t)$ fails to match the granny target $(t^2-t+1)^2$. These are eliminated by Filter 2.
\end{itemize}

In every case, the elimination is unconditional: at no non-negative integer parameter value does any of the $83$ candidate braids satisfy both filters simultaneously. This completes the homological elimination of the hyperbolic case in the proof of Theorem \ref{thm:main_detection}.
