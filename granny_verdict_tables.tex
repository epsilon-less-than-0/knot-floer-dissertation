\chapter{Verdict Tables for the Granny Detection}\label{app:granny_verdict_tables}

This appendix tabulates the homological filtering verdicts for all $83$ candidate braid families generated by the train-track analysis of Chapters \ref{chap:stratoriented} and \ref{chap:stratatraintrackanalysis}, evaluated against the granny target Alexander polynomial $\Delta(t) = (t^2-t+1)^2$. The data here support the proof of Theorem \ref{thm:main_detection}, specifically the conclusions of Theorems \ref{thm:candelabra_stratum_classification} and \ref{thm:two_triangle_stratum_classification}.

For each family, we list:
\begin{itemize}
    \item the family name (as recorded in Appendices \ref{app:candidate_braids_candelabra}--\ref{app:candidate_braids_track_17}),
    \item the determinant $\det(I - A_{\text{red}})$, expressed as a constant or a polynomial in the family's twist parameters $k, m, j$,
    \item the verdict, which records the means of elimination.
\end{itemize}

The verdicts are categorized as follows:
\begin{description}
    \item[\textbf{F1}] Filter 1 fails outright: $\det(I - A_{\text{red}})$ is a constant other than $\pm 1$, so the open book does not bound an integer homology sphere.
    \item[\textbf{F1$^{*}$}] Filter 1 fails parametrically: $\det(I-A_{\text{red}})$ is a non-constant polynomial in the family parameters, and no assignment of non-negative integer values to those parameters yields $|\det| = 1$.
    \item[\textbf{F2}] Filter 1 admits at least one non-negative integer parameter value where $|\det(I - A_{\text{red}})| = 1$, but at every such value the characteristic polynomial $\det(tI - A_{\text{red}})$ fails to match the granny target $(t^2-t+1)^2$.
\end{description}

All verdicts were computed by direct evaluation of the matrices $M_1, \dots, M_5$ from Section \ref{sec:homological_filtering}, using the parsing rules and execution pipeline described in the implementation of the homological filter. The reduced action is invariant under conjugation, full twists, and mirroring (Section \ref{sec:homological_filtering}), so testing a single representative braid word per family suffices. For parametric families that pass Filter 1 at some non-negative integer parameter value (verdict F2), the parameter search was conducted over the ranges $k, m \in [0, 14]$ (two parameters) or $k, m, j \in [0, 9]$ (three parameters), which exhaustively covers the small-twist solutions; the characteristic polynomial obstruction at those solutions then closes the case.

\section{Summary}

Across all $83$ candidate families:
\begin{itemize}
    \item $42$ families are eliminated by Filter 1 with $\det(I-A_{\text{red}})$ a constant other than $\pm 1$ (verdict F1).
    \item $20$ families are eliminated by Filter 1 parametrically: the determinant is a polynomial in the family parameters that admits no non-negative integer assignment giving $|\det| = 1$ (verdict F1$^{*}$).
    \item $21$ families admit non-negative integer parameter values at which $|\det(I-A_{\text{red}})| = 1$, but at every such value the characteristic polynomial $P(t)$ fails to match the granny target $(t^2-t+1)^2$ (verdict F2). Of these, $2$ are constant cases ($\det = -1$ for the Jubilant family on Track 0 and the Boke family on Track 17, both yielding $P(t) = t^4 - 3t^2 + 1$), and $19$ are parametric.
\end{itemize}

In every case, the elimination is unconditional: at no non-negative integer parameter value does any of the $83$ candidate braids satisfy both filters simultaneously. This completes the homological elimination of the hyperbolic case in the proof of Theorem \ref{thm:main_detection}.
